Политика конфиденциальности

1. Общие положения

Настоящая политика конфиденциальности регулирует порядок обработки и использования персональных данных пользователей.

2. Какие данные собираются

Пользователь предоставляет следующие данные:

  • имя
  • адрес электронной почты

3. Цели сбора данных

Персональные данные используются для:

  • связи с пользователем
  • предоставления информации об услугах
  • обработки заявок

4. Передача данных третьим лицам

Для сбора данных используется сторонний сервис (Tally). Данные могут обрабатываться этим сервисом в рамках его функционала.

5. Защита данных

Оператор принимает необходимые меры для защиты персональных данных от неправомерного доступа и распространения.

6. Согласие пользователя

Пользователь, оставляя данные через форму на сайте, выражает согласие с данной политикой конфиденциальности.

7. Контакты

По всем вопросам обработки персональных данных пользователь может связаться по email: systembiohacking@gmail.com
